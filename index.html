    • TexasVoice Web Application
    This is a web application for the election of the Student Council President (Ketua OSIS) at SMKS Texar Karawang. The application is named "TexasVoice", and it is designed to facilitate the students of SMKS Texar Karawang in voting for the Student Council President for the new academic year. The application has two user roles: admin and voter. The admin has the ability to manage student, teacher, and Student Council President candidate data, as well as view the overall voting results. Voters, on the other hand, can cast their votes or choose from the available Student Council President candidates. It is developed using the following tools :
  • PHP Language (Codeigniter 4)
  • Bootstrap (AdminLTE 3)
  • MySQL Database
  • XAMPP Control Panel
  • Visual Studio Code
  • Web Browser (Microsoft Edge)
  • PhpSpreadsheet Library

  • Create, Read, Update, Delete Data
  • Import File (.xlsx)
  • Login & Logout
  • Upload Image
  • Total Vote Chart
