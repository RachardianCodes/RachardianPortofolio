TexasVoice

TexasVoice is a web-based application designed specifically for the election of the student council president at SMKS Texar Karawang. The application provides two levels of access, namely admin and voter, each with different functions according to their respective roles.

Key Features:

Admin Panel:
  • Voter Data Management: Admin can manage voter data, including registration and deletion of voters.

  • Candidate Management: Admin can add, edit, or delete data for student council president candidates.

  • Vote Result Monitoring: Admin has access to view real-time election results.
