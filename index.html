    • TexasVoice Web Application
    "TexasVoice" is a web application developed for SMKS Texar Karawang's Student Council President election. It serves two user roles: admin and voter. Admins manage student, teacher, and candidate data, and view voting results. Voters can cast votes for Student Council President candidates. The application aims to streamline the election process for the new academic year. It is developed using the following tools :
  • PHP Language (Codeigniter 4)
  • Bootstrap (AdminLTE 3)
  • MySQL Database
  • XAMPP Control Panel
  • Visual Studio Code
  • Web Browser (Microsoft Edge)
  • PhpSpreadsheet Library
